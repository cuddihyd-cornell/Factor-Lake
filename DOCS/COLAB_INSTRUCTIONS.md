## Troubleshooting

### Ngrok tunnel not working?
- Make sure you signed up at https://dashboard.ngrok.com/
- Copy the ENTIRE token (it's long)
- Restart the runtime if needed: **Runtime â†’ Restart runtime**

### Installation taking too long?
- First run takes 2-3 minutes (installing packages)
- Subsequent runs are faster (~30 seconds)

### Connection errors?
- Check your internet connection
- Try refreshing the page
- Restart the runtime and run again

### Can't access the URL?
- The ngrok URL is public - share it with anyone!
- URL expires when you stop the Colab session
- Run the notebook again to get a new URL
